# 如何构建CSS
目标:
1. CSS作用到HTML上
2. 用CSS对文字进行格式化

## 在你的HTML里面应用CSS
三种方式:
1. 外部样式表
2. 内部样式表
3. 内联样式

### 外部样式表
使用<link>引用外部css文件, 可以使用相对路径

### 内部样式表
HTML文档里面用style标签, 少数情况这么做

### 内联样式
在元素的属性上使用style属性, 尽量不用这样做,难以维护
